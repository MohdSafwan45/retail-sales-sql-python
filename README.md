# Retail Sales Analysis — SQL + Python

## Project Overview
End-to-end data analysis of retail sales data using SQL (SQLite) and Python.
Extracted actionable business insights on revenue trends, customer demographics,
and product category performance.

## Tools & Technologies
- **Python** — Pandas, Matplotlib, Seaborn
- **SQL** — SQLite (via Python)
- **Jupyter Notebook**
- **Dataset** — Retail Sales Dataset (Kaggle)

## Key Insights
- Electronics is the top revenue-generating category
- Age group 25–40 has highest average spend per transaction
- Peak sales occur in Q4 (Oct–Dec)
- Top 10 customers contribute ~15% of total revenue
